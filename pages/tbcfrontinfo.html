Front-end Development

რეგისტრაცია დაიწყება 2025 წელს

კურსის აღწერა

Frontend დეველომპენტი გულისხმობს ვებ გვერდის /ვებ აპლიკაციის user interface-ს და user experience-ს შექმნას და იმპლემენტაციას. Frontend დეველოპერი ქმნის იმ ვიზუალურ ელემენტებს, რომლებთანაც მომხმარებლებს პირაპირი შეხება აქვთ თავიანთი ბრაუზერის გამოყენებით.

Frontend დეველოპმენტის კურსის მიზანია სტუდენტების აღჭურვა იმ ცოდნითა და უნარებით, რომლებიც საჭიროა კომპლექსური ვებ გვერდების და აპლიკაციების ასაწყობად.

კურსის ფარგლებში სტუდენტი შეისწავლის პროგრამულ ენებს და ინსტრუმენტებს, როგორიცაა: Bash, Git, HTML, CSS, SCSS/SASS, JavaScript, TypeScript, Angular.

კურსის დასრულების შემდგომ სტუდენტს შესაძლებლობა აქვს დასაქმდეს Agile გუნდში Frontend დეველოპერის პოზიციაზე და სრულფასოვნად ჩაერთოს რეალურ დეველოპმენტის პროცესებში.

კურსის გავლის შემდეგ შეგეძლება:

დამოუკიდებლად ააწყო კომპლექსური SPA ვებ აპლიკაციები Angular Framework-ის საშუალებით
შექმნა ვიზუალურად მიმზიდველი, Responsive მომხმარებლის ინტერფეისი
აპლიკაციის აწყობისას გამოიყენო სხვადასხვა მნიშვნელოვანი ოპტიმიზაციის ტექნიკები
გატესტო აპლიკაცია სრულფასოვანი Unit/Integration ტესტების საშუალებით
ეფექტურად გამოიყენო ბრაუზერის developer tools
შენს აპლიკაციებში გამოიყენო სხვადასხვა საინტერესო და გამოსადეგი third party ბიბლიოთეკები
გამოიყენო უახლესი Version Control ხელსაწყოები

ვისთვისაა კურსი?

კურსის სამიზნე აუდიტორია მოიცავს 18 წლიდან ინდივიდების ფართო სპექტრს, რომლებიც ისწრაფვიან ისწავლონ Frontend Development-ი.


მათ შორისაა: კომპიუტერული მეცნიერების სტუდენტები, თვითნასწავლი დეველოპერები, Career Switchers (პროფესიონალები, რომლებიც ამჟამად არიან სხვა სფეროებში, მაგრამ აქვთ დიდი ინტერესი გადართონ თავიანთი კარიერა Frontend Development-ის მიმართულებით), არატექნიკური პირები (ადამიანები, რომლებსაც არ აქვთ ფორმალური ტექნიკური განათლება, მაგრამ აქვთ დიდი სურვილი და მოტივაცია ისწავლონ Frontend Development-ი).

კურსის წინაპირობა: JavaScript პროგრამული ენის საფუძვლები, OOP-ის პრინციპების თეორიული ცოდნა.

კურსის შინაარსი

Git

ვერსიების კონტროლის ყველაზე ფართოდ გამოყენებადი ხელსაწყოს შესწავლა, რომელიც გვეხმარება მარტივი ბრძანებების საშუალებით ვმართოთ ჩვენი რეპოზიტორიები.

HTML

HTML5 მარქაფ ენისშესწავლა საბაზისო დონიდან მაღალ დონემდე.

CSS/SASS/SCSS

Cascading StyleSheets ენის და მისი preprocessor-ების სრულყოფილად შესწავლა საბაზისო დონიდან მაღალ დონემდე.

Javascript

ყველაზე პოპულარული პროგრამული ენის, Javascript-ის შესავალა საბაზისო დონიდან მაღალ დონემდე.

TypeScript

TypeScript პროგრამირების ენა, მისი დამატებითი შესაძლებლობების შესწავლა/გამოყენება.

NPM/Yarn

Package Manager-ების გამოყენება

Angular

Angular Framework-ის შესწავლა, რომლის საშუალებითაც შეგვიძლია ავაწყოთ კომპლექსური Single Page ვებ აპლიკაციები.

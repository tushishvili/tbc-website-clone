ლექციების დაწყება: ოქტომბრის თვეში

Backend development ნიშნავს ისეთი პროგრამული უზრუნველყოფის შექმნას, რომელიც მოემსახურება ვებ გვერდს და პასუხისმგებელი იქნება ყველაფერზე, რაც კი შეიძლება არსებობდეს სერვერის მხარეს. კურსზე პლატფორმად შემოთავაზებულია .Net Core რომელიც არის Microsoft-ის მიერ შექმნილი Open Source Framework-ი. იგი გამოიყენება სხვადასხვა სახის აპლიკაციის შესაქმნელად, იქნება ეს Web, Mobile, Desktop, Game და ა.შ. ასევე, .Net Core არის Cross-platform, რაც ნიშნავს რომ ის შეგვიძლია გავუშვათ Windows, Linux, MacOs ოპრაციულ სისტემებზე.

.NET კურსის მიზანია მრავალმხრივი სწავლებისა და პრაქტიკული გამოცდილების გაღრმავება Backend Development-ის ძირითად კონცეფციებზე, პროგრამულ ენა C#-ზე, .NET Core framework-ზე.

კურსის დასრულების შემდგომ სტუდენტეს შეეძლება ეჯაილ გუნდში Junior Backend Developer-ად მუშაობა. კურსის გავლის შემდეგ შეგეძლება:

ვისთვისაა კურსი?

კურსის სამიზნე აუდიტორია მოიცავს 18 წლიდან ინდივიდების ფართო სპექტრს, რომლებიც ისწრაფვიან ისწავლონ Backend Development .NET Core Framework-ის საშუალებით.

მათ შორისაა: დამწყები დეველოპერები, კომპიუტერული მეცნიერების სტუდენტები, უმცროსი დეველოპერები, თვითნასწავლი დეველოპერები, Career Switchers (პროფესიონალები, რომლებიც ამჟამად არიან სხვა IT სფეროებში, მაგრამ აქვთ დიდი ინტერესი გადართონ თავიანთი კარიერა Backend Development-ის მიმართულებით), არატექნიკური პირები (ადამიანები, რომლებსაც არ აქვთ ფორმალური ტექნიკური განათლება, მაგრამ აქვთ დიდი სურვილი და მოტივაცია ისწავლონ Backend Development).

კურსის წინაპირობა: პროგრამირების საფუძვლები, OOP-ის პრინციპების თეორიული ცოდნა

ვინ მასწავლის?

კურსის ტრენერი - ლაშა დაუშვილი

ლაშა თბილისის სახელმწიფო უნივერსიტეტის კომპიუტერული მეცნიერებების საბაკალავრო/სამაგისტრო პროგრამის კურსდამთავრებულია. იგი მე-9 წელია, რაც TBC ბანკის ჯგუფის წევრია და ამჟამად Backend დეველოპერის/Chapter Lead-ის პოზიციას იკავებს. პარალელურად ლაშა უკვე 4 წელია TBC IT აკადემიის .NET (Backend Development) კურსის ტრენერია. მას ასევე მიჰყავს SQL Server-ის შიდა კურსი TBC ბანკში, ისევე როგორც TBC ბანკის რისკების აკადემიაში. ბოლო 4 წელიწადში ლაშას 1000-ზე მეტი ადამიანის პროფესიულ გადამზადებაში აქვს მონაწილეობა მიღებული.

კურსის ტრენერი - ბექა ღვაბერიძე

ბექა თბილისის სახელმწიფო უნივერსიტეტის კომპიუტერული მეცნიერებების საბაკალავრო და ინფორმაციული ტექნოლოგიების სამაგისტრო პროგრამების კურსდამთავრებულია. მისი კარიერული წინსვლა IT სფეროში ჯერ კიდევ 2012 წლიდან სტაჟიორის პოზიციაზე იწყება. 2012 წლიდან დღემდე ბექა დასაქმებული იყო არაერთ კომპანიაში ისეთ პოზიციებზე, როგორიც არის უფროსი ინჟინერი, ტექნიკური Team Lead, დეველოპმენტის ხელმძღვანელი, დეპარტამენტის უფროსი, CTO, პროგრამული უზრუნველყოფის არქიტექტორი და ა.შ. პარალელურად ბექა თავის მრავალწლიან სამუშაო გამოცდილებას TBC IT აკადემიაში .NET (Backend Development) კურსზე აზიარებს.

კურსის ტრენერი - დიმიტრი დონდოლაძე

დიმიტრი სან-დიეგოს სახელმწიფო უნივერსიტეტის კომპიუტერული მეცნიერების კურსდამთავრებულია. უნივერსიტეტის გარეთ დიმიტრიმ 2019 წელს დაასრულა TBC IT აკადემიის .NET (Backend Development) კურსი, რის შემდგომაც მან თავისი კარიერა TBC ბანკში .NET დეველოპერის პოზიციაზე განაგრძო. TBC IT აკადემიაში მიღებული გამოცდილება დიმიტრიმ ისეთ კომპანიებში გააზიარა როგორიც არის Epam, DataArt. ამჟამად იგი დასაქმებულია კომპანია BP-ის ლონდონის ოფისში Backend Application Engineer-ის პოზიციაზე. თავის საერთაშორისო გამოცდილებას დიმიტრი TBC IT აკადემიაში .NET (Backend Development) კურსზე, მენტორის პოზიციაზე აზიარებს.

ხშირად დასმული კითხვები

რა არის თიბისი IT აკადემიაში სწავლის ღირებულება?

თიბისი IT აკადემიის აბსოლუტურად ყველა კურსი 100%-ით დაფინანსებულია თიბისი ბანკის მიერ. შესაბამისად, კურსზე წარმატებით ჩარიცხვის შემთხვევაში სტუდენტს არ უწევს არანაირი საფასურის გადახდა.

რამდენ ხანს გრძელდება სწავლა თითოეულ კურსზე?

თიბისი IT აკადემიის კურსების ხანგრძლივობა მერყეობს სამი თვიდან ექვს თვემდე. დეტალური ინფორმაცია კურსის ვადების შესახებ შეგიძლიათ იხილოთ უშუალოდ კურსების აღწერის გვერდზე.

რამდენი სტუდენტის მიღება ხდება კურსზე?

კურსზე ადგილების რაოდენობა განსხვავებულია და მერყეობს 20-დან 30-მდე.

რა ეტაპებია გასავლელი კურსზე მოსახვედრად?

იმისათვის, რომ სტუდენტი ჩაირიცხოს თიბისი IT აკადემიის რომელიმე კურსზე საჭიროა შემდეგი ეტაპების გავლა:
1. რეგისტრაცია ვებ გვერდზე მითითებულ ვადებში
2. უნარების ონლაინ ტესტირება
3. ტექნიკური ტესტირება
4. HR სამოტივაციო ვიდეოს ატვირთვა
5. ტექნიკური გასაუბრება
ეტაპები შეიძლება განსხვავდებოდეს კურსის მიხედვით, მაგალითად, ზოგ შემთხვევაში, ამოღებულია უნარების ტესტირება. შესაძლოა ტექნიკური ტესტირება ჩანაცვლებული იყოს ტექნიკური დავალებით და ა.შ.

შემიძლია თუ არა რამოდენიმე კურსზე რეგისტრაცია?

დიახ, სტუდენტს შეუძლია დარეგისტრირდეს რამოდენიმე კურსზე და მიიღოს მონაწილეობა ერთზე მეტი კურსის შერჩევაში. თუმცა, აკადემიის შიდა პოლიტიკიდან გამომდინარე, აპლიკანტს შეუძლია მხოლოდ და მხოლოდ ერთ კურსზე სწავლა. შესაბამისად, იმ შემთხვევაში თუ აპლიკანტი ორი ან მეტი კურსის შერჩევის ეტაპს წარმატებით გაივლის მას მოუწევს აირჩიოს კონკრეტულად რომელი მიმართულებით უნდა გააგრძელოს სწავლა.

აკადემიის კურსებზე სწავლა ონლაინ რეჟიმში ტარდება თუ ფიზიკურად?

თიბისი IT აკადემიაში სწავლა ტარდება სრულიად ონლაინ.

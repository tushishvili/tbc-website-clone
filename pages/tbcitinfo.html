ლექციების დაწყება: ოქტომბრის თვეში

Android Development არის პროცესი, რომელიც გულისხმობს აპლიკაციების აგებას Android ოპერაციული სისტემისთვის. იგი შესაძლებლობას აძლევს დეველოპერებს, შექმნან მრავალფეროვანი აპლიკაციები, განავითარონ თამაშები, სოციალური კავშირების აპლიკაციები, e-commerce პლატფორმები და სხვა.
კურსის მიზანია უზრუნველყოს მრავალმხრივი სწავლება და პრაქტიკული გამოცდილების გაღრმავება Android Development-ის ძირითად კონცეფციებზე, პროგრამულ ენა Kotlin-ზე.
კურსის დასრულების შემდგომ სტუდენტეს შეეძლება საკუთარი აპლიკაციის არქიტექტურისა და გადაწყვეტების განსაზღვრა, ამოცანის დასმა და მისი გადაჭრისთვის საჭირო რესურსების მოკვლევა/განსაზღვრა.

ვისთვისაა კურსი?

კურსის სამიზნე აუდიტორია მოიცავს 18 წლიდან ინდივიდების ფართო სპექტრს, რომლებიც ისწრაფვიან ისწავლონ Android აპლიკაციების დეველოპმენტი.

მათ შორისაა: დამწყები დეველოპერები, კომპიუტერული მეცნიერების სტუდენტები, უმცროსი დეველოპერები, თვითნასწავლი დეველოპერები, Career Switchers (პროფესიონალები, რომლებიც ამჟამად არიან სხვა IT სფეროებში, მაგრამ აქვთ დიდი ინტერესი გადართონ თავიანთი კარიერა Android Development მიმართულებით), არატექნიკური პირები (ადამიანები, რომლებსაც არ აქვთ ფორმალური ტექნიკური განათლება, მაგრამ აქვთ დიდი სურვილი და მოტივაცია ისწავლონ Android Development).

კურსის წინაპირობა: პროგრამირების საფუძვლები, OOP-ის პრინციპები

კურსის შინაარსი

შესავალი Android Development-ში
Android ოპერაციული სისტემის მიმოხილვა, აპლიკაციების ასაგებად Საჭირო ხელსაწყოებისა და ტექნოლოგიების განხილვა

Android studio
Android studio-ს შესაძლებლობები და მასში არსებული ხელსაწყოების გამოყენება

Kotlin პროგრამირების ენა
Kotlin პროგრამირების ენის საფუძვლიანი შესწავლა საბაზისო დონიდან მაღალ დონემდე

Android აპლიკაციების შექმნა
Android-ში SDK დეტალური განხილვა. Android აპლიკაციისათვის საჭირო ხელსაწყოების/ფრეიმორკების განხილვა და პრაქტიკაში გამოყენება

დიზაინ პატერნები და არქიტექტურა
დიზაინ პატერნების განხილვა და შესწავლა MVC, MVP, MVVM და MVI. Clean Architecture მნიშვნელობა და პრაქტიკაში დანერგვა
Networking და Data Persistence
Http Request და Http Response Retrofit-ის გამოყენებით. Room Database-სა და Data Store-ში მონაცემების დამუშავება

Kotlin coroutines Android-ში
ასინქრონული პროცესების მართვა coroutines-ის გამოყენებით

Dependency injection Android-ში
არსებული ფრეიმორკების განხილვა და პრაქტიკაში Dagger hilt-ის დანერგვა

Android Background Work Overview
WorkManager-ის საფუძვლიანი განხილვა და პრაქტიკაში გამოყენება

Jetpack Compose და მისი პერსპექტივა
Android აპლიკაციის აგება Jetpack Compose-ის გამოყენებით

ტესტირება Android-ში
Android-ში ტესტირების ტიპებისა და საჭირო ხელსაწყოების განხილვა

ვინ მასწავლის?

კურსის ტრენერი - ნიკა ტაბატაძე

ნიკა თბილისის სახელმწიფო უნივერსიტეტის კომპიუტერული მეცნიერებების საბაკალავრო, შემდგომ კი სამაგისტრო პროგრამის კურსდამთავრებულია. ნიკა 8 წელზე მეტია, რაც Android developer-ია. ამჟამად იგი გერმანულ კომპანია Syniotec-ში Mobile Development Team Lead-ის პოზიციაზეა დასაქმებული. თავის საერთაშორისო გამოცდილებას ნიკა უკვე მესამე წელია, რაც TBC IT აკადემიის Android Development კურსზე ტრენერის პოზიციაზე აზიარებს. TBC IT აკადემიის გარეთ ნიკა ასევე კითხულობდა Android Development კურსს ბიზნესისა და ტექნოლოგიების უნივერსიტეტში.

კურსის ტრენერი - ლუკა ფარჩუკიძე

ლუკა ბიზნესისა და ტექნოლოგიების უნივერსიტეტის ინფორმაციული ტექნოლოგიების საბაკალავრო პროგრამის კურსდამთავრებულია. იგი 3 წელია უკვე, რაც პროგრამირების სფეროშია დასაქმებული. უნივერსიტეტის პარალელურად ლუკამ 2020 წელს წარჩინებით გაიარა TBC IT აკადემიის კურსი Android Development-ის მიმართულებით, რის შემდგომაც TBC ბანკის გუნდის წევრი გახდა. მან თავისი აკადემიური საქმიანობა ჯერ კიდევ 2021 წელს USAID X BTU Android კურსის ფარგლებში დაიწყო, სადაც მენტორის პოზიციაზე იყო დასაქმებული, ხოლო შემდგომ კი TBC IT აკადემიაში გადმოერთო აქტიურად. ამჟამად ლუკა დასაქმებულია კომპანია Syniotec-ში, სადაც მუშაობს Google-ის ახალ ტექნოლოგია KMP-ზე (Kotlin Multiplatform).

ხშირად დასმული კითხვები

რა არის თიბისი IT აკადემიაში სწავლის ღირებულება?

თიბისი IT აკადემიის აბსოლუტურად ყველა კურსი 100%-ით დაფინანსებულია თიბისი ბანკის მიერ. შესაბამისად, კურსზე წარმატებით ჩარიცხვის შემთხვევაში სტუდენტს არ უწევს არანაირი საფასურის გადახდა.

რამდენ ხანს გრძელდება სწავლა თითოეულ კურსზე?

თიბისი IT აკადემიის კურსების ხანგრძლივობა მერყეობს სამი თვიდან ექვს თვემდე. დეტალური ინფორმაცია კურსის ვადების შესახებ შეგიძლიათ იხილოთ უშუალოდ კურსების აღწერის გვერდზე.

რამდენი სტუდენტის მიღება ხდება კურსზე?

კურსზე ადგილების რაოდენობა განსხვავებულია და მერყეობს 20-დან 30-მდე.

რა ეტაპებია გასავლელი კურსზე მოსახვედრად?

იმისათვის, რომ სტუდენტი ჩაირიცხოს თიბისი IT აკადემიის რომელიმე კურსზე საჭიროა შემდეგი ეტაპების გავლა:
1. რეგისტრაცია ვებ გვერდზე მითითებულ ვადებში
2. უნარების ონლაინ ტესტირება
3. ტექნიკური ტესტირება
4. HR სამოტივაციო ვიდეოს ატვირთვა
5. ტექნიკური გასაუბრება
ეტაპები შეიძლება განსხვავდებოდეს კურსის მიხედვით, მაგალითად, ზოგ შემთხვევაში, ამოღებულია უნარების ტესტირება. შესაძლოა ტექნიკური ტესტირება ჩანაცვლებული იყოს ტექნიკური დავალებით და ა.შ.

შემიძლია თუ არა რამოდენიმე კურსზე რეგისტრაცია?

დიახ, სტუდენტს შეუძლია დარეგისტრირდეს რამოდენიმე კურსზე და მიიღოს მონაწილეობა ერთზე მეტი კურსის შერჩევაში. თუმცა, აკადემიის შიდა პოლიტიკიდან გამომდინარე, აპლიკანტს შეუძლია მხოლოდ და მხოლოდ ერთ კურსზე სწავლა. შესაბამისად, იმ შემთხვევაში თუ აპლიკანტი ორი ან მეტი კურსის შერჩევის ეტაპს წარმატებით გაივლის მას მოუწევს აირჩიოს კონკრეტულად რომელი მიმართულებით უნდა გააგრძელოს სწავლა.

აკადემიის კურსებზე სწავლა ონლაინ რეჟიმში ტარდება თუ ფიზიკურად?

თიბისი IT აკადემიაში სწავლა ტარდება სრულიად ონლაინ.
